# Panda Login + Signup Page

A fully responsive and animated Login and Signup UI developed by Avdesh Jadon.
This project features a panda theme with smooth animations, focus interactions, and clean form transitions.

**Live Demo:** [View it live](https://avdeshjadon.github.io/PandaLogin-SignUpPage)

## Features

- Responsive Login & Signup forms
- Panda animation
- Smooth input focus effects
- Toggle between Login and Signup
- Password input visibility toggle
- Mobile-friendly design
- Hosted via GitHub Pages

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages

## How to Run Locally

```bash
git clone https://github.com/avdeshjadon-dev/PandaLoginPage.git
cd PandaLoginPage
open index.html
```
